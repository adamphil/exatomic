Gaussian Orbitals¶

gaussian.js¶

GaussianOrbital¶

The necessary machinery required to generate GaussianOrbitals is laid out in the functions order_gtf_basis and construct_mos. This is merely a thin wrapper around the AtomicField class.

construct_mos¶

From a list of properly formatted strings representing basis functions, take appropriate linear combinations of them weighted by MO coefficients to generate the full string representation of the MOs.

order_gtf_basis¶

Unpack the nested data structures to generate strings representing basis functions. This necessarily assumes some knowledge of the way a computational code orders basis functions, therefore it will not work for all cases and a different function may be needed.
